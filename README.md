# Blog.Backend

### Git First Push.
Use the package manager [composer](https://getcomposer.org/) to install foobar.

## Git Clone.

```bash
git clone https://github.com/psmever/blog.backend.git blog.backend
```

## Composer.
```bash
composer install

```

## First Config.
```bash
composer install
cp .env.example .env
```
